Routes


GET api/concerts

Route qui liste tous les concerts de la base de données


GET api/concerts/:id

Route qui affiche un concert en fonction de son ID:
exemple: api/concerts/11129131 affichera le concert de B.B Jacques


GET api/concerts

Route qui liste tous les concerts de la base de données


GET api/concerts/:date

Route qui affiche tous les concert à une certaine date:
exemple: api/concerts/2025-04-03 affiche les concerts du 3 avril 2025

Route qui liste tous les users de la base de données
